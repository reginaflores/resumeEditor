# Resume Editor

A minimal, browser-based resume editor for Regina Flores Mir.

## Features

- **Inline editing** — click "Enable Editing" to edit any field directly in the browser
- **Print to PDF** — use the "Save as PDF" button to export a clean, print-optimized PDF
- **No dependencies** — single HTML file, no build step required

## Usage

Open `index.html` in any modern browser. No server needed.

## Stack

- HTML / CSS / Vanilla JavaScript
- [DM Serif Display & DM Sans](https://fonts.google.com/) (Google Fonts)
